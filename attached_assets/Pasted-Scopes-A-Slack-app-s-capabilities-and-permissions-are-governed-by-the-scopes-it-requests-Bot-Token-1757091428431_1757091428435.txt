Scopes
A Slack app's capabilities and permissions are governed by the scopes it requests.

Bot Token Scopes
Scopes that govern what your app can access.

OAuth Scope
Description
 
app_mentions:read
View messages that directly mention @SSELFIE STUDIO in conversations that the app is in

assistant:write
Allow SSELFIE STUDIO to act as an App Agent

canvases:read
SSELFIE STUDIO will be able to access contents of canvases created inside Slack.

channels:history
View messages and other content in public channels that SSELFIE STUDIO has been added to

channels:read
View basic information about public channels in a workspace

chat:write
Send messages as @SSELFIE STUDIO

chat:write.customize
Send messages as @SSELFIE STUDIO with a customized username and avatar

chat:write.public
Send messages to channels @SSELFIE STUDIO isn't a member of

conversations.connect:read
Receive Slack Connect invite events sent to the channels SSELFIE STUDIO is in

conversations.connect:write
Create Slack Connect invitations for channels that SSELFIE STUDIO has been added to, and accept invitations sent to SSELFIE STUDIO

files:read
View files shared in channels and conversations that SSELFIE STUDIO has been added to

files:write
Upload, edit, and delete files as SSELFIE STUDIO

groups:history
View messages and other content in private channels that SSELFIE STUDIO has been added to

groups:read
View basic information about private channels that SSELFIE STUDIO has been added to

groups:write
Manage private channels that SSELFIE STUDIO has been added to and create new ones

im:history
View messages and other content in direct messages that SSELFIE STUDIO has been added to

im:read
View basic information about direct messages that SSELFIE STUDIO has been added to

im:write
Start direct messages with people

links:read
View URLs in messages

reactions:read
View emoji reactions and their associated content in channels and conversations that SSELFIE STUDIO has been added to

reminders:read
View reminders created by SSELFIE STUDIO

reminders:write
Add, remove, or mark reminders as complete

remote_files:read
View remote files added by the app in a workspace

users:read
View people in a workspace

workflows.templates:read
Manage Slack workflow template on user's behalf

workflows.templates:write
Write Slack workflow template on user's behalf
