# EditorialStory Component

A reusable editorial-style story section with image and text content, following the SSELFIE design system.

## Usage

```tsx
import EditorialStory from '@/components/sections/editorial/EditorialStory'

<EditorialStory
  headline="Your main headline"
  paragraphs={[
    "First paragraph of your story.",
    "Second paragraph with <br /> line breaks if needed.",
    "Third paragraph continues the narrative."
  ]}
  imageSrc="/path/to/your/image.jpg"
  imageAlt="Descriptive alt text"
/>
```

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `headline` | `string` | ✅ | - | The main headline text (Times New Roman) |
| `paragraphs` | `string[]` | ✅ | - | Array of paragraph content (supports HTML) |
| `imageSrc` | `string` | ✅ | - | Image source for the left side |
| `imageAlt` | `string` | ✅ | - | Alt text for the image |
| `backgroundColor` | `string` | ❌ | `#f5f5f5` | Background color override |
| `textColor` | `string` | ❌ | `#0a0a0a` | Text color override |
| `reversed` | `boolean` | ❌ | `false` | Reverse layout (image right, text left) |
| `button` | `object` | ❌ | - | Optional button configuration |
| `button.text` | `string` | ✅ | - | Button text |
| `button.href` | `string` | ✅ | - | Button link destination |
| `button.variant` | `'primary' \| 'secondary'` | ❌ | `'primary'` | Button style variant |

## Design System Compliance

- ✅ **Colors**: Uses Editorial Gray (#f5f5f5) and Editorial Black (#0a0a0a)
- ✅ **Typography**: Times New Roman for headlines, Inter for body text
- ✅ **Spacing**: Consistent with style guide (py-20 md:py-32)
- ✅ **Responsive**: Mobile-first design with proper breakpoints
- ✅ **Accessibility**: Proper alt text and semantic HTML

## Examples

### With Button (Pricing Page)
```tsx
<EditorialStory
  headline="Okay, here's what actually happened…"
  paragraphs={[
    "This didn't start as a business.<br />It started as survival.",
    "One year ago, I hit rock bottom. Divorced. Three kids. No backup plan.",
    "And one day, in the middle of all that mess—I picked up my phone."
  ]}
  imageSrc={SandraImages.editorial.laptop1}
  imageAlt="Sandra working, not posed"
  button={{
    text: "Read my full story",
    href: "/about",
    variant: "secondary"
  }}
/>
```

### Reversed Layout

```tsx
<EditorialStory
  headline="The transformation begins"
  paragraphs={["Your story content here..."]}
  imageSrc="/image.jpg"
  imageAlt="Transformation image"
  reversed={true}
/>
```

### Custom Colors

```tsx
<EditorialStory
  headline="Special section"
  paragraphs={["Content..."]}
  imageSrc="/image.jpg"
  imageAlt="Special image"
  backgroundColor="#ffffff"
  textColor="#0a0a0a"
/>
```

### With Primary Button

```tsx
<EditorialStory
  headline="Call to action section"
  paragraphs={["Your content..."]}
  imageSrc="/image.jpg"
  imageAlt="CTA image"
  button={{
    text: "Get started",
    href: "/studio",
    variant: "primary"
  }}
/>
```

## Notes

- Paragraphs support HTML content via `dangerouslySetInnerHTML` for line breaks
- Image is optimized with Next.js Image component
- Fully responsive with proper image sizing
- Follows luxury editorial design principles
